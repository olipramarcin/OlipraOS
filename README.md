## Description 📜

OlipraOS is a portfolio project built for learning purposes, focused on connecting a web interface with a Python backend through an interactive browser-based terminal.

The application simulates a basic operating system environment where users can enter commands and receive responses processed on the server side. The goal of this project is to explore client–server communication, request handling, and dynamic UI updates in a practical way.

## Features 🔥

- Interactive terminal in the browser
- Basic command handling
- Real-time communication with a backend API
- Dynamic output rendering without page reloads

## Future Plans ⭐

- Expanding the command system and backend logic
- Adding simple games (e.g., Snake) runnable from the terminal
- Evolving the interface into a more advanced OS-like environment (Windows-style UI)
- Introducing a system for managing apps launched from the terminal

### Purpose 🤔

The goal of OlipraOS is not to create a full operating system, but to build a solid foundation for experimenting with web application architecture and backend–frontend integration.

## How To Use 🎈

1. Launch the application using: <br>
`python app.py`
2. Open the app in your browser (localhost:5000).
3. Use the terminal interface:
- Type a command
- Press Enter
- View the response directly in the terminal output
4. Experiment with available commands and observe how the frontend communicates with the backend in real time.

#### Installation 📲

1. Clone the repository: <br>
`git clone <your-repo-url>` <br>
`cd OlipraOS`
2. Install the required dependencies: <br>
`pip install flask`
3. Run the application: <br>
`python app.py`
4. Open your browser and go to: <br>
` http://127.0.0.1:5000`
